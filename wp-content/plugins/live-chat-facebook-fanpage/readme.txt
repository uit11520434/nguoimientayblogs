== Description ==

Add a chat to your website, using the messenger and your Facebook fan page.

Finally you can create your customer service is completely free and you can respond to the demands of your users directly from smartphones and tablets, without having to pay any fee to third party services.

== Installation ==


1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->FB Live Chat to use the plugin


== Frequently Asked Questions ==

= Can  I use all the Facebook fan pages? =

You must be the administrator of the page, otherwise you cannot chat with users.

= How do I add my page to the plugin? =

You just add the URL of your Facebook page (e.g.: https://www.facebook.com/your-fan-page)

= How can I use this chat on my smartphone? =
